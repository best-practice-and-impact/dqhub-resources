Data quality resources

In the Government Data Quality Hub we are developing resources to help people with their data quality work.

These resources are currently in development:

  • Data Quality Practices Matrix V1 - A matrix to assess current data quality practice and identify areas for improvement
  • Data Quality Evidence Base - A collection of evidence that can help to support a business case for data quality work

A draft template that can be used or adapted for recording data quality action plans

These resources support our Data Quality Action Plan training. They're simplified to illustrate lessons from the training package and shouldn't be taken as best practice measurements for similar datasets.

  • Example design of a data quality action plan for a hypothetical dataset
  • Example results of a data quality action plan on a hypothetical dataset

These resources are designed for use in public sector organisations.
If you have any feedback on these resources, please email DQHub@ons.gov.uk
